## Madphp

Madphp framework.

## usage method 1

### Download:

```bash
git clone https://github.com/xiaominge/madphp.git
cd madphp
```

### Install:

```bash
composer install
```

## usage method 2

```bash
composer require madphp/madphp : dev-master
```
